PROJECT #4 Note Taking WebPage

This note-taking webpage is a modern, responsive application designed with a glassmorphism effect for a frosted-glass aesthetic. It allows users to add, edit, and delete notes, with all data saved in the browser’s localStorage for persistence. The design has been updated with improved styling, featuring smooth animations, hidden scrollbars that appear on hover, and a clean layout using CSS Grid. The app includes a popup modal for managing notes, dynamic note rendering via JavaScript, and intuitive controls for editing or deleting entries. It works seamlessly on both desktop and mobile devices, offering a sleek and user-friendly experience.
